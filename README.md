# Skin Cancer AI Detection Web Application

## Description

This project is a Flask-based web application that uses a pre-trained VGG16 deep learning model to classify skin lesions as Benign or Malignant.

## Features

* User authentication
* Image upload
* Skin lesion analysis
* AI prediction using VGG16
* Patient history storage in MySQL
* Result visualization

## Technologies

* Python
* Flask
* TensorFlow
* MySQL
* Bootstrap 5
